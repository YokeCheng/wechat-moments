# 爆了么 - 内容智能生产平台

## 1. 项目描述
面向公众号/头条号创作者的一站式内容生产工具。核心主链路：
**每日爆款 → 提示词库 → 智能生文 → 一键排版 → 公众号发布**

目标用户：个人创作者、MCN机构、品牌运营人员。

## 2. 页面结构

### 一级导航（顶部）
- `/` - 每日爆款（默认落地页）
- `/prompts` - 提示词库
- `/writer` - 智能生文
- `/layout` - 一键排版
- `/channels` - 公众号管理
- `/tutorial` - 使用教程
- `/contact` - 联系我们
- `/vip` - 会员中心

### 每日爆款子模块（Tab切换）
- 公众号爆款
- 头条号爆款
- 今日热搜榜

## 4. 数据模型（Mock阶段）
- 文章：id, title, field, platform, views, likes, shares, publishTime, author
- 提示词：id, title, category, content, tags, createdAt
- 公众号：id, name, avatar, followers, authorized
